# ISSTA23-Artifacts

The parser and the symbolic execution engine of Eunomia are uploaded into the `parser` folder and `engine` folder, respectively.

Please mention that some prerequisites are necessary for using parser and engine:
- Ubuntu 18.04 and Python 3.8;
- Packages:
  - Antlr runtime 4.9.2 ([link](https://www.cs.upc.edu/~padro/CL/practica/install.html))
  - Boost ([link](https://stackoverflow.com/questions/12578499/how-to-install-boost-on-ubuntu))
  - WABT ([link](https://github.com/WebAssembly/wabt))
